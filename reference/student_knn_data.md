# Student KNN Example Data for practicing knntuner functions

A simulated student dataset for practicing K-nearest neighbors tuning
and estimation

## Usage

``` r
student_knn_data
```

## Format

A data frame with 100 rows and 6 variables:

- id:

  Student identification number

- age:

  Student age in years

- study_hours:

  Number of study hours per week

- score:

  Test score

- private_tutoring:

  Whether the student received private tutoring

- pass:

  Whether the student passed based on the score threshold of at least 70

## Source

Simulated dataset created for examples in this package
